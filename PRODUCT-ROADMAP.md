# Tripboard 功能大纲

这个文档用于防止后续迭代遗忘和跑偏。每完成一项，就在这里同步勾选并补充状态。

## 状态说明

- `[x]` 已完成并已推送公网
- `[ ]` 未完成
- `[~]` 部分完成，仍有后续依赖或升级项

## 功能清单

### 1. 交通票价

- [~] 官方查询入口
  - 已完成：携程查询、Trip.com 查询、12306 查询入口。
  - 已完成：根据当前出发地、目的地、日期生成查询关键词。
- [~] 手动保存报价
  - 已完成：可手动保存航班/动车班次、时间、价格到当天交通列表。
  - 已完成：手动报价会进入协作计划并同步。
- [~] Supabase 交通代理
  - 已完成：前端可调用 Supabase Edge Function。
  - 当前限制：无正式 Trip.com/Ctrip API 权限时只返回代理示例。
- [ ] 真实 Trip.com/Ctrip 机票/动车 API
  - 未完成原因：需要具体产品 API 权限、AppKey/AppSecret、接口文档。
  - 当前判断：个人账号大概率无法直接申请完整机票/火车票 API。

### 2. AI 优化路径

- [~] 本地路径排序
  - 已完成：按地点坐标/经纬度做简单距离排序。
- [~] AI 代理接口入口
  - 已完成：可填写 AI 路径代理地址和访问令牌。
- [ ] 真实 AI 路线规划
  - 未完成：尚未接入模型后端。
  - 待实现：考虑营业时间、实时交通、排队、天气、体力、停留时长。

### 3. 高德地图定位

- [~] 高德搜索链接
  - 已完成：可生成高德搜索链接。
- [~] 经纬度录入
  - 已完成：地点可填写经纬度。
- [~] 高德代理接口入口
  - 已完成：可填写高德地点代理地址。
- [ ] 高德地图 SDK
  - 未完成：未嵌入地图 SDK。
- [ ] 真实地点搜索/路线规划 API
  - 未完成：需要高德 Web 服务 Key 或后端代理。

### 4. 外部 App 同步

- [~] 手动导入
  - 已完成：美团/点评、民宿/酒店、截图/分享可手动录入。
- [~] 文本解析
  - 已完成：粘贴订单文本后可尝试解析名称、时间、金额、地址。
- [ ] 自动读取外部 App 订单
  - 未完成：需要平台授权/API。
  - 当前限制：不能直接读取用户手机 App 或账号订单。

### 5. 多人协作冲突处理

- [x] 实时同步
  - 已完成：Supabase Realtime 同步计划。
- [x] 历史版本/恢复
  - 已完成：编辑前、本地/远端更新前保存历史版本。
- [ ] 字段级合并
  - 未完成：当前仍是整份计划 JSON 保存。
- [ ] 编辑锁/正在编辑提示
  - 未完成：尚未做到地点级锁定或冲突提示。

### 6. 成员权限

- [ ] 管理员/成员角色

  - 已完成：按地点预算汇总。

  - 已完成：可填写自有天气代理。

## 后续优先级建议

1. 先完成 AA 结算/转账建议，因为不依赖第三方平台授权。
2. 再做地点级编辑锁，降低多人同时编辑覆盖风险。
3. 再接高德地图 SDK，因为能显著提升路线和定位体验。
4. 真实携程/Trip.com 票价 API 等拿到企业权限后再接。
